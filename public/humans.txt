# Ritvik Goyal - Software Developer Portfolio

This portfolio website showcases the professional work and skills of Ritvik Goyal, a software developer based in Toronto, Canada. Built with modern web technologies and optimized for performance and search engine visibility.

- **Frontend**: React 18, TypeScript, Material-UI
